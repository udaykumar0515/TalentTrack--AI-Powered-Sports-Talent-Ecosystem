## Guidelines

- Use PNG format for quality
- Maintain consistent aspect ratio (preferably 16:9 or similar)
- Use browser zoom to ensure consistent sizing
- Hide browser URL bar and extensions for cleaner screenshots
- Use dark/light mode consistently across screenshots
- Ensure sensitive data is not visible (use demo accounts)
